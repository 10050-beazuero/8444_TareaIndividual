JSON

Concepto

JSON es un formato de archivo que utiliza texto legible por humanos para almacenar y transmitir objetos de datos que contienen matrices y pares de atributos y valores. JSON se utiliza para almacenar información de manera organizada y de fácil acceso. JSON significa Notación de objetos de JavaScript. Ofrece una colección de datos legibles por humanos a los que se puede acceder lógicamente.

Ejemplo JSON: 
'{"name":"John", "age":30, "car":null}'

Características de JSON
  • Fácil de usar : la API JSON ofrece una fachada de alto nivel, lo que lo ayuda a simplificar los casos de uso más comunes.
  • Rendimiento : JSON es bastante rápido ya que consume mucho menos espacio de memoria, lo que es especialmente adecuado para sistemas o gráficos de objetos grandes.
  • Herramienta gratuita : la biblioteca JSON es de código abierto y de uso gratuito.
  • JSON limpio : crea un resultado JSON limpio y compatible que es fácil de leer.
  • Los datos están en pares de nombre/valor

Ejemplo

El siguiente ejemplo muestra cómo crear un objeto y acceder a un objeto JavaScript como este:

Codigo


<!DOCTYPE html>
<html>
<body>

<h2>Access a JavaScript object</h2>
<p id="demo"></p>
<button onclick="myFunction()">Pruebalo</button>
<script>
function myFunction() {
const myObj = {name:"John", age:30, city:"New York"};
document.getElementById("demo").innerHTML = myObj.name +","+myObj.age+","+myObj.city;
}
</script>

</body>
</html>

JSON .parse()

Un uso común de JSON es intercambiar datos hacia/desde un servidor web. Al recibir datos de un servidor web, los datos siempre son una cadena. Analice los datos con JSON.parse(), y los datos se convierten en un objeto de JavaScript. Cuando se usa JSON.parse()en un JSON derivado de una matriz, el método devolverá una matriz de JavaScript, en lugar de un objeto de JavaScript.

Ejemplo

Los objetos de fecha no están permitidos en JSON.El siguiente ejemplo muestra cómo convierte una cadena en una fecha:

Codigo


<!DOCTYPE html>
<html>
<body>

<h2>Convierta una cadena en un objeto de fecha.</h2>
<p id="demo2"></p>
<button onclick="fecha()">Pruebalo</button>
<script>
function fecha(){
const text = '{"name":"John", "birth":"2023-01-03", "city":"New York"}';
const obj = JSON.parse(text);
obj.birth = new Date(obj.birth);
document.getElementById("demo2").innerHTML = obj.name + ", " + obj.birth; 
}
</script>

</body>
</html>
